# ポートフォリオ用Railsアプリケーション

## 使用技術(予定)
- ユーザー認証機能
  - devise
- テンプレート言語
  - Slim
- 検索機能
  - ransack
- ページネーション機能
  - kaminari
- 画像投稿機能
  - CarrierWave
- 日本語化
  - i18n
- テスト環境
  - Rspec
- フロントエンド
  - React

## 依存関係
- Rails: 5.1.7
- Ruby: 2.6.1p33
